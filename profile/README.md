## 📖Rules
* 매주 1회 대면(혹은 비대면) 스터디: **매주 목요일 오후 4시 30분**
  * 장소: 학교 도서관 스터디룸 or 인근 카페
  * 당일날 **랜덤**으로 **각 문제**마다 코드 리뷰 발표자를 정함
* 과제 미제출 or 스터디 결석 시 벌금
  * 피치 못할 상황 발생 시 미리 연락할 것
  * 최소 전주 주말까지 연락하기 
  * 사전연락 없이 미제출/결석 시 벌금
      * 결석: 4000원
      * 연락 늦음: 2000원
      * 과제 미제출: 5000원
      * 과제 미흡: 문제 1개당 1000원
